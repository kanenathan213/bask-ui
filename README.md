# Bask: Estimate how much time to spend in the sun

### Install dependencies:
```
yarn install
```

If you haven't done so yet, install Elm globally:
```
yarn install -g elm
```

Install Elm's dependencies:
```
elm package install
```

### Serve locally:
```
yarn start
```

### Build & bundle for production:
```
yarn run build
```

* Files are saved into the `/dist` folder
* To check it, open `dist/index.html`


### Supported platforms

Desktop/Android Chrome 53<
iOS Safari 9<
